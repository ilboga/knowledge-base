# Wiki Index

This is the entry point for the LLM Wiki — a compiled, agent-maintained knowledge base.
It catalogs every concept and source in the wiki. It is read first, every session, and
updated in the same commit as any wiki change.

**Scope:** a broad, general-purpose personal knowledge base spanning many domains —
work, research, learning, and ideas. The wiki is intentionally not pre-partitioned into
fixed categories; topic groupings under **Concepts** emerge organically as sources are
ingested. When a cluster of related concepts grows large enough to deserve its own
heading, add one here.

For how the wiki works and how to add sources, see [README.md](README.md). For the full
schema and operating rules, see [CLAUDE.md](CLAUDE.md).
